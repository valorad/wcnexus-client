About wcnexus.com

5 years of development, with new stuff coming in (almost) each year

WCNexus started being developed back in 2016, and almost every year, it was implemented with more beautiful page layouts and built with newer technologies.

The online version is v4, which was developed in 2021. The backend is powered by .Net 5, while the front-end is based on Angular 11.

The website is hosted on a VPS, and has been deployed via docker and is served with HTTPS enabled since v3. I know it seems nothing fancy nowadays, but hey, it was the first time that I've managed to figure all these things out myself!
